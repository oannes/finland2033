# Input relevance — which teams are briefed on which injects

Each team sees only the inputs relevant to its seat (information asymmetry:
you infer the rest of the system from what it does, not from a shared
omniscient brief). Injects marked `(all)` — and any inject not listed — are
briefed to everyone. The facilitator always sees everything.
